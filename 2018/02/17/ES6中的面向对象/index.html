一、ES6中类的声明方式

注意点

  1. 类的声明并没有变量提升
  2. 类的声明里只能有方法，没有属性
  3. 支持表达式方式 var Foo = class { … };
  4. 类声明里的方法是不可枚举的，不能被遍历
