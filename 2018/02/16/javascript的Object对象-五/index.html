javascript的Object对象--实例方法(五)

发表于 2018-02-16

一、hasOwnProperty()

判断对象的某个属性是自身属性还是原型链上的属性。

二、isPrototypeOf()

判断某个原型对象是否在某个对象的原型链上，跟instanceof的功能相同，只是调用的方向不一样。

三、propertyIsEbumerable()

判断自身属性是否是可枚举的,是hasOwnProperty的加强版

四、toString()

返回当前对象的字符串形式

比typeof判断变量类型更加具体的方式

五、valueOf()

返回当前对象对应的值
