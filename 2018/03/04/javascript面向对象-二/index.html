new运算符做了什么

  1. 创建一个新对象，使该对象的proto指向构造函数的prototype
  2. 让this指向该对象
  3. 执行构造函数
  4. 判断构造函数的返回值是引用类型，如果是就返回该值，如果不是就返回新创建的对象
